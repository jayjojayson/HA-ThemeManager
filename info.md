# Theme Manager for Home Assistant

A custom integration that provides a curated collection of high-quality themes as YAML files in `/config/themes/`. Theme selection works directly in the HA user profile — no dashboard card, no Lovelace setup needed.

## Included Themes

15 hand-crafted themes with full **Auto / Dark / Light** support:

- **Modern Dark** — Google Material inspired
- **Dracula** — Classic Dracula color scheme
- **Catppuccin Mocha** — Catppuccin Mocha / Latte
- **Solarized Dark** — Solarized Dark / Light
- **GitHub Dark** — GitHub Dark / Light
- **Nord** — Nord Polarnight / Snowstorm
- **Monokai** — Monokai Dark / Light
- **Oceanic Dark** — Deep ocean tones
- **HA HomeDashboard** — Minimalist transparent cards (requires Card-Mod)
- **Glass** — Semi-transparent blur effect cards (requires Card-Mod)
- **Sharp** — 0 px radius flat design
- **Aurora** — Northern lights gradient
- **Horizon** — Sunset gradient
- **Mono** — Terminal green monospace
- **Rounded** — Large radii, soft pastels

## Installation via HACS

1. Add this repository as a custom repository in HACS
2. Install **Theme Manager**
3. Restart Home Assistant
4. Go to **Settings → Devices & Services → Add Integration → Theme Manager**
5. Choose your theme under **My Account → Appearance → Theme**
